В этой демонстрации мы с помощью командой строки выполняем - создание репозитория - добавление ориджина - пуш изменений – пул изменений – добавление ориджина - ----- Создадим новый проект Стартуем там гит с помощью git init Теперь добавим файл chapter1.html Глава 1 И вот однажды с собачкой случилась необычная история. \ День был солнечный и как раз случился выходной. Она бежала по дорожке и виляла хвостиком, так как ей было хорошо. Но в один момент собачка побежала в другую сторону. Она виляла хвостом и бежала пока не прибежала на окраину города. Добавим файлы $ cd project $ git add chapter1.html Сделаем коммит $ git commit -m "сообщение коммита" ---- Говорим : Следующий шаг потребует от нас авторизоваться на Github и получить токен Заходим на Github в настройки Ставим нужные галочки и создаем токен Копируем токен В пайчарме делаем пуш, получаем запрос авторизации Вводим логин и токен вместо пароля Видим что все работает! ---- Теперь отправим наши изменения в удаленный репозиторий $ git push Перейдем в гитхаб и увидим что все хорошо --- Теперь снова отправимся на гитхаб и изменим один из файлов, chapter1, удалив оттуда половину текста Теперь изменения нужно подхватить локально. Для этого мы выполним команду $ git pull Отлично! Мы видим, что изменения подтянулись --- ------ А теперь и вовсе перейдем в системы и склонируем удаленный репозиторий $ git clone и мы видим что у нас появились все файлы Перетащим папку в пичарм – и можем работать как раньше
