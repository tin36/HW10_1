Задача 11

Вам необходимо реализовать приложение с представлением index, работающим по урлу /search/.

При GET-запросе это представление должно отображать форму с текстовым полем с именем search.

При отправке формы на сервер методом POST, сервер возвращает html-документ с текстом Результаты по запросу search, который отображается как заголовок первого уровня, а значение search - курсивом.
